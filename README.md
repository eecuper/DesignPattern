# 23种设计模式   

## 创建型

#### [工厂模式](doc/FactoryPattern.md)   

当类的创建步骤比较复杂，需要创建多个对象。将类的创建步骤交由工厂类实现，我们只要创建工厂类就可以。

#### [建造者模式](doc/Build.md)   

创建建造流程相同，属性内容、顺序不同的复杂对象。

#### [单例模式](doc/singleton .md)

在单例模式下，对象只创建一次。

#### [原型模式](doc/prototype.md)  

通过一个对象克隆出其他对象。
